# Contributing to SafeSpace AI

We welcome contributions from the community. Please follow these guidelines to contribute to the project.

## How to Contribute

1.  Fork the repository.
2.  Create a new branch for your feature or bug fix.
3.  Make your changes and commit them with a descriptive commit message.
4.  Push your changes to your fork.
5.  Create a pull request to the `main` branch of this repository.

## Code Style

We use ESLint and Prettier to enforce a consistent code style. Please make sure to run the linter before creating a pull request.

```sh
npm run lint
```

## Reporting Bugs

If you find a bug, please create an issue in the GitHub repository. Please include a clear and concise description of the bug, and steps to reproduce it.
